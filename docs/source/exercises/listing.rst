Exercise Reference
===================

This page provides a listing of the exercises that are currently supported by **Cryptoserve**.
Each exercise is designed to highlight a specific cryptographic concept.

.. danger::
   The following exercises are designed as interactive client-server exchanges that simulate cryptographic processes.
   While some protocols may be based on real cryptographic ciphers and algorithms, they are often modified or used in a way
   to demonstrate a particular concept. These exercises are **not** cryptographically secure and are intended **solely for educational purposes**.
